Privacy Policy

Last updated: 18-01-2024

Introduction

Encrypted Password Saver is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and safeguard your personal information.

Information We Collect

Encrypted Password Saver does not collect any personal information from users. The app uses fingerprint authentication for user access, and no additional data is requested or stored.

Authentication

Encrypted Password Saver utilizes fingerprint authentication for user login. This information is securely stored on the user's device and is not transmitted or stored externally.

Passwords

Users have the option to set passwords for additional login security. These passwords are securely stored using encryption techniques and remain on the user's device.

Permissions

Encrypted Password Saver does not request any permissions from users. The app operates solely with the user's authentication data stored on the device.

Security

We prioritize the security of your data. Encrypted Password Saver employs robust encryption methods to protect user authentication information.

Changes to this Privacy Policy

We may update this Privacy Policy from time to time. Users will be notified of any significant changes through the app or our website.
